## 介绍

一个简单的待办事项列表，一个端到端的待办事项列表 dapp，从智能合约端开始到前端，最后使用钱包与两者进行交互

## 使用

连接钱包，在事项窗口中输入事项名称，点击add，付费后事项列表中就出现该事项。完成事项后，在该事项上打勾，无需付费，即可标注事项完成。
